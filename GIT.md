# Comandos de Git

Para clonar un repositorio usamos
```
git clone <url>
```

Para subir un archivo o archivos a nuestro repositorio remoto, seguimos el metodo ACP

Para verificar que archivos son nuevos o tienen cambios, ejecutamos un 

```
git status
```
